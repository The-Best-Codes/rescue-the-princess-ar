## OVERVIEW

An AR game controlled by hand tracking and facial expressions. Rescue the princess from a dragon using your body as the controller.

## INPUT METHODS

- **Hand Tracking** - Move and interact with gesture controls
- **Facial Recognition** - Trigger actions with expressions
- **Augmented Reality** - Collect coins and defeat the dragon in augmented reality
